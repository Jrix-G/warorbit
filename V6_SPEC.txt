SECTION 1 — RÈGLES DU JEU (référence pour le simulateur)
================================================================================

1.1 TERRAIN
- Plateau 100×100, soleil au centre (50,50), rayon 10
- 2 ou 4 joueurs (en compétition : 4 joueurs)
- 500 tours maximum
- Victoire : joueur avec le plus de ships TOTAL (planètes + flottes en vol)

1.2 PLANÈTES
- Statiques ou en orbite autour du soleil
- Orbite si : distance_centre + rayon < 50 (ROT_LIMIT)
- Position orbitale : angle_initial + angular_velocity * tour
- Production : 1-5 ships/tour si propriétaire != -1 (neutre = 0)
- owner = -1 → neutre, owner = 0..3 → joueur

1.3 FLOTTES
- Déplacement en ligne droite vers angle donné
- Vitesse logarithmique : speed(n) = 1 + 5 * (log(n)/log(1000))^1.5
  → 1 ship = 1 u/tour, 1000 ships ≈ 6 u/tour
- Capture de planète : flotte arrive si distance < rayon_planète
- Combat à l'arrivée :
  * Si même owner → renfort (addition)
  * Si owner différent :
    - attaquant > défenseur → planète capturée, ships = attaquant - défenseur
    - attaquant < défenseur → défense tient, ships = défenseur - attaquant
    - égalité → planète neutre, 0 ships

1.4 COMÈTES
- Apparaissent aux tours 50, 150, 250, 350, 450
- Restent 50 tours
- Se déplacent à 4 u/tour sur une trajectoire fixe
- Capturables comme une planète mobile
- Production pendant la durée de possession

1.5 ACTIONS PAR TOUR
- Format : [[planet_id, angle_radians, n_ships], ...]
- Un ordre par planète source maximum
- n_ships doit être <= ships disponibles sur la planète

1.6 BUDGET TEMPS
- 1 seconde par tour (actTimeout)
- remainingOverageTime : budget supplémentaire global (s'épuise)
- Il faut rendre la réponse en <0.9s pour sécurité

3.1 REPRÉSENTATION DE L'ÉTAT

class GameState:
    """
    Représentation compacte de l'état du jeu.
    Tout en numpy pour vectorisation maximale.
    """
    # Planètes : array shape (N_planets, 7)
    # Colonnes : [id, owner, x, y, radius, ships, production]
    planets : np.ndarray  # float32

    # Flottes : array shape (N_fleets, 8)
    # Colonnes : [id, owner, x, y, angle, from_planet_id, ships, progress]
    # progress = distance parcourue / distance totale [0,1]
    fleets : np.ndarray   # float32

    # Positions initiales des planètes (pour orbites)
    # Shape (N_planets, 2) : [x_init, y_init]
    init_positions : np.ndarray

    # Vitesse angulaire globale (scalaire)
    angular_velocity : float

    # IDs des planètes-comètes actives
    comet_ids : set

    # Tour actuel
    step : int

    # Joueur local (pour self-play)
    player : int

NOTE IMPLÉMENTATION :
- Utiliser float32 partout (2× plus rapide que float64 sur numpy)
- Pré-allouer les arrays au maximum (max 20 planètes, max 200 flottes)

3.2 FONCTION DE MISE À JOUR ORBITALE

def update_planet_positions(state: GameState) -> None:
    """
    Met à jour x,y des planètes en orbite.
    Vectorisé numpy, O(N_planets).
    """
    dx = state.init_positions[:, 0] - 50.0
    dy = state.init_positions[:, 1] - 50.0
    r = np.hypot(dx, dy)

    # Masque : planètes en orbite
    in_orbit = r + state.planets[:, 4] < 50.0  # radius col

    angle_init = np.arctan2(dy, dx)
    angle_now = angle_init + state.angular_velocity * state.step

    state.planets[in_orbit, 2] = 50.0 + r[in_orbit] * np.cos(angle_now[in_orbit])
    state.planets[in_orbit, 3] = 50.0 + r[in_orbit] * np.sin(angle_now[in_orbit])

3.3 FONCTION DE VITESSE (vectorisée)

def fleet_speeds(ships_array: np.ndarray) -> np.ndarray:
    """Vitesse log pour array de flottes. Vectorisé."""
    speeds = np.ones_like(ships_array, dtype=np.float32)
    mask = ships_array > 1
    log_ships = np.log(np.maximum(ships_array[mask], 1.0))
    speeds[mask] = 1.0 + 5.0 * (log_ships / np.log(1000.0)) ** 1.5
    return speeds

3.4 FONCTION DE STEP PRINCIPAL

def step(state: GameState, actions_by_player: dict) -> GameState:
    """
    Simule 1 tour complet. Retourne nouvel état (copie).

    actions_by_player : {player_id: [[from_id, angle, ships], ...]}

    Ordre des opérations (conforme au moteur Kaggle) :
    1. Production sur planètes possédées
    2. Lancement des nouvelles flottes (depuis les actions)
    3. Déplacement de toutes les flottes
    4. Résolution des combats (flottes arrivées)
    5. Mise à jour positions orbitales
    6. Gestion comètes (si tour = 50/150/250/350/450)
    """
    new_state = state.copy()  # copie shallow + arrays copiés
    new_state.step += 1

    # 1. Production
    owned = new_state.planets[:, 1] >= 0
    new_state.planets[owned, 5] += new_state.planets[owned, 6]

    # 2. Lancement flottes
    new_fleets = []
    for player_id, actions in actions_by_player.items():
        for from_id, angle, n_ships in actions:
            planet_idx = np.where(new_state.planets[:, 0] == from_id)[0]
            if len(planet_idx) == 0:
                continue
            pidx = planet_idx[0]
            available = int(new_state.planets[pidx, 5])
            n = min(n_ships, available)
            if n <= 0:
                continue
            new_state.planets[pidx, 5] -= n
            # Ajouter flotte : [id, owner, x, y, angle, from_id, ships, 0]
            px, py = new_state.planets[pidx, 2], new_state.planets[pidx, 3]
            new_fleets.append([len(new_state.fleets) + len(new_fleets),
                               player_id, px, py, angle, from_id, n, 0.0])

    if new_fleets:
        new_state.fleets = np.vstack([new_state.fleets,
                                       np.array(new_fleets, dtype=np.float32)])

    # 3. Déplacement flottes (vectorisé)
    if len(new_state.fleets) > 0:
        speeds = fleet_speeds(new_state.fleets[:, 6])
        new_state.fleets[:, 2] += np.cos(new_state.fleets[:, 4]) * speeds
        new_state.fleets[:, 3] += np.sin(new_state.fleets[:, 4]) * speeds

    # 4. Résolution combats (flottes arrivées sur planètes)
    _resolve_arrivals(new_state)

    # 5. Positions orbitales
    update_planet_positions(new_state)

    return new_state

3.5 RÉSOLUTION DES ARRIVÉES

def _resolve_arrivals(state: GameState) -> None:
    """
    Pour chaque planète, vérifie les flottes arrivées.
    Résout les combats. Supprime les flottes résolues.
    """
    if len(state.fleets) == 0:
        return

    to_remove = []

    for pidx in range(len(state.planets)):
        px, py, pradius = state.planets[pidx, 2], state.planets[pidx, 3], state.planets[pidx, 4]
        planet_owner = int(state.planets[pidx, 1])

        # Flottes arrivées sur cette planète
        dists = np.hypot(state.fleets[:, 2] - px, state.fleets[:, 3] - py)
        arrived_mask = dists < pradius
        arrived_indices = np.where(arrived_mask)[0]

        if len(arrived_indices) == 0:
            continue

        # Grouper par owner
        for fidx in arrived_indices:
            fleet_owner = int(state.fleets[fidx, 1])
            fleet_ships = int(state.fleets[fidx, 6])

            if fleet_owner == planet_owner:
                # Renfort
                state.planets[pidx, 5] += fleet_ships
            else:
                # Combat
                planet_ships = int(state.planets[pidx, 5])
                if fleet_ships > planet_ships:
                    state.planets[pidx, 1] = fleet_owner
                    state.planets[pidx, 5] = fleet_ships - planet_ships
                elif fleet_ships < planet_ships:
                    state.planets[pidx, 5] = planet_ships - fleet_ships
                else:
                    state.planets[pidx, 1] = -1
                    state.planets[pidx, 5] = 0
                planet_owner = int(state.planets[pidx, 1])

            to_remove.append(fidx)

    if to_remove:
        mask = np.ones(len(state.fleets), dtype=bool)
        mask[to_remove] = False
        state.fleets = state.fleets[mask]

def state_from_obs(obs) -> GameState:
    """
    Convertit l'observation Kaggle en GameState numpy.
    Appelé 1 fois par tour au début.
    """
    planets = np.array(obs.get("planets", []), dtype=np.float32)
    fleets_raw = obs.get("fleets", [])

    if fleets_raw:
        fleets = np.array(fleets_raw, dtype=np.float32)
        # Ajouter colonne progress (=0 pour flottes en vol)
        progress = np.zeros((len(fleets), 1), dtype=np.float32)
        fleets = np.hstack([fleets, progress])
    else:
        fleets = np.zeros((0, 8), dtype=np.float32)

    init_planets = np.array(obs.get("initial_planets", []), dtype=np.float32)
    init_positions = init_planets[:, 2:4] if len(init_planets) > 0 else planets[:, 2:4].copy()

    return GameState(
        planets=planets,
        fleets=fleets,
        init_positions=init_positions,
        angular_velocity=obs.get("angular_velocity", 0.0),
        comet_ids=set(obs.get("comet_planet_ids", [])),
        step=obs.get("step", 0),
        player=obs.get("player", 0)
    )

def generate_candidates(state: GameState, me: int, n_candidates: int) -> list:
    """

    - 20% : plans agressifs (envoyer 70-80% vers planètes ennemies proches)

    for i in range(n_candidates):
        strategy = i % 5
        actions = _generate_action_set(state, me, strategy)
        candidates.append(actions)

    return candidates

4.3 SIMULATION ET SCORING

def beam_search(state: GameState, time_budget: float = 0.85) -> list:
    """
    Algorithme principal de beam search.

    Returns: liste d'actions [[from_id, angle, ships], ...]
    """
    import time
    t_start = time.time()
    me = state.player

    # Génération adaptative selon temps disponible
    t_elapsed = time.time() - t_start
    n_candidates = min(100, max(20, int(50 * time_budget / 0.85)))
    horizon = min(25, max(10, int(20 * time_budget / 0.85)))

    candidates = generate_candidates(state, me, n_candidates)

    best_score = -np.inf
    best_actions = []

    # Adversaire simulé : utilise heuristique V5 pour les autres joueurs
    other_players = [p for p in range(4) if p != me and
                     np.any(state.planets[:, 1] == p)]

    for i, my_actions in enumerate(candidates):
        if time.time() - t_start > time_budget * 0.9:
            break  # Sécurité temps

        sim_state = state.copy()
        score = 0.0
        discount = 1.0
        GAMMA = 0.97  # Discount futur

        for t in range(horizon):
            # Actions de tous les joueurs
            all_actions = {me: my_actions}
            for opp in other_players:
                all_actions[opp] = _heuristic_actions(sim_state, opp)

            sim_state = step(sim_state, all_actions)

            # Score intermédiaire à chaque tour (avec discount)
            turn_score = evaluate_state(sim_state, me) * discount
            score += turn_score
            discount *= GAMMA

            # Early termination si victoire/défaite claire
            if _is_terminal(sim_state):
                break

        # Score final
        score += evaluate_state(sim_state, me) * discount * 5.0  # Bonus état final

        if score > best_score:
            best_score = score
            best_actions = my_actions

    return best_actions

4.4 SIMULATION DES ADVERSAIRES (heuristique V5)

def _heuristic_actions(state: GameState, player: int) -> list:
    """
    Simule les actions des adversaires avec heuristique simple.
    Doit être très rapide (appelé des centaines de fois).
    Version simplifiée de V5 : cibler le meilleur ratio valeur/distance.
    """
    my_planets = state.planets[state.planets[:, 1] == player]
    if len(my_planets) == 0:
        return []

    actions = []
    targets = state.planets[state.planets[:, 1] != player]
    if len(targets) == 0:
        return []

    for planet in my_planets:
        ships = int(planet[5])
        if ships < 5:
            continue

        # Trouver meilleure cible par valeur/distance
        px, py = planet[2], planet[3]
        dists = np.hypot(targets[:, 2] - px, targets[:, 3] - py)
        values = targets[:, 6] * 30 - dists * 0.5  # production * horizon - coût distance
        best_idx = np.argmax(values)

        target = targets[best_idx]
        angle = np.arctan2(target[3] - py, target[2] - px)
        n_send = int(ships * 0.6)
        actions.append([int(planet[0]), angle, n_send])

    return actions

5.2 FEATURES D'ENTRÉE (24 dimensions)

def extract_features(state: GameState, me: int) -> np.ndarray:
    """
    Extrait 24 features normalisées depuis un GameState.
    Invariant au nombre de planètes/flottes.
    """
    features = np.zeros(24, dtype=np.float32)

    # Ships totaux par joueur (normalisés)
    total_ships = np.zeros(4)
    for p in range(4):
        planet_ships = np.sum(state.planets[state.planets[:, 1] == p, 5])
        fleet_ships = np.sum(state.fleets[state.fleets[:, 1] == p, 6]) if len(state.fleets) > 0 else 0
        total_ships[p] = planet_ships + fleet_ships

    my_ships = total_ships[me]
    total = total_ships.sum() + 1e-8

    features[0] = my_ships / total                          # Part de ships
    features[1] = my_ships / (max(total_ships) + 1e-8)     # Ratio vs leader

    # Planètes
    my_planets = state.planets[state.planets[:, 1] == me]
    all_planets = len(state.planets)

    features[2] = len(my_planets) / max(all_planets, 1)    # Part planètes
    features[3] = np.sum(my_planets[:, 6]) / max(all_planets * 3, 1)  # Production relative

    # Menaces entrantes
    incoming_threat = 0
    if len(state.fleets) > 0:
        enemy_fleets = state.fleets[state.fleets[:, 1] != me]
        for ef in enemy_fleets:
            for mp in my_planets:
                d = np.hypot(ef[2] - mp[2], ef[3] - mp[3])
                if d < 30:
                    incoming_threat += ef[6]

    features[4] = incoming_threat / (my_ships + 1e-8)      # Niveau de menace

    # Phase de jeu
    features[5] = state.step / 500.0                       # Progression temporelle
    features[6] = 1.0 if state.step < 40 else 0.0         # Early game
    features[7] = 1.0 if 40 <= state.step < 150 else 0.0  # Mid game
    features[8] = 1.0 if state.step >= 150 else 0.0       # Late game

    # Comètes
    features[9] = 1.0 if len(state.comet_ids) > 0 else 0.0
    comet_owned = sum(1 for cid in state.comet_ids
                      if np.any((state.planets[:, 0] == cid) & (state.planets[:, 1] == me)))
    features[10] = comet_owned / max(len(state.comet_ids), 1)

    # Position relative des adversaires
    enemy_ships_sorted = sorted([total_ships[p] for p in range(4) if p != me], reverse=True)
    for i, s in enumerate(enemy_ships_sorted[:3]):
        features[11 + i] = s / total  # Ships des 3 adversaires triés

    # Concentration géographique (défense vs expansion)
    if len(my_planets) > 1:
        centroid_x = np.mean(my_planets[:, 2])
        centroid_y = np.mean(my_planets[:, 3])
        spread = np.mean(np.hypot(my_planets[:, 2] - centroid_x,
                                   my_planets[:, 3] - centroid_y))
        features[14] = spread / 70.0  # Normaliser par demi-diagonale

    # Distance au soleil (vulnérabilité trajectoires)
    if len(my_planets) > 0:
        sun_dists = np.hypot(my_planets[:, 2] - 50, my_planets[:, 3] - 50)
        features[15] = np.min(sun_dists) / 70.0

    # Tours restants features
    remaining = (500 - state.step) / 500.0
    features[16] = remaining
    features[17] = 1.0 if remaining < 0.1 else 0.0  # Endgame

    # Ratio ships en vol vs total
    my_fleet_ships = np.sum(state.fleets[state.fleets[:, 1] == me, 6]) if len(state.fleets) > 0 else 0
    features[18] = my_fleet_ships / (my_ships + 1e-8)

    # Production potentielle (si on capture toutes les neutres proches)
    neutral_planets = state.planets[state.planets[:, 1] == -1]
    if len(neutral_planets) > 0 and len(my_planets) > 0:
        centroid_x = np.mean(my_planets[:, 2])
        centroid_y = np.mean(my_planets[:, 3])
        neutral_dists = np.hypot(neutral_planets[:, 2] - centroid_x,
                                  neutral_planets[:, 3] - centroid_y)
        close_neutral_prod = np.sum(neutral_planets[neutral_dists < 40, 6])
        features[19] = close_neutral_prod / 20.0

    # Features 20-23 : réservées pour extensions futures

    return features

5.3 ARCHITECTURE DU RÉSEAU

class NumpyEvaluator:
    """
    Réseau numpy 3 couches. Inférence en ~0.05ms.
    """
    def __init__(self, input_dim=24, hidden1=32, hidden2=16):
        # Initialisation He
        self.W1 = np.random.randn(input_dim, hidden1).astype(np.float32) * np.sqrt(2/input_dim)
        self.b1 = np.zeros(hidden1, dtype=np.float32)
        self.W2 = np.random.randn(hidden1, hidden2).astype(np.float32) * np.sqrt(2/hidden1)
        self.b2 = np.zeros(hidden2, dtype=np.float32)
        self.W3 = np.random.randn(hidden2, 1).astype(np.float32) * np.sqrt(2/hidden2)
        self.b3 = np.zeros(1, dtype=np.float32)

    def predict(self, features: np.ndarray) -> float:
        """Forward pass. Features shape (24,). Retourne probabilité [0,1]."""
        h1 = np.maximum(0, features @ self.W1 + self.b1)  # ReLU
        h2 = np.maximum(0, h1 @ self.W2 + self.b2)        # ReLU
        out = h2 @ self.W3 + self.b3                       # Linear
        return float(1.0 / (1.0 + np.exp(-out[0])))        # Sigmoid

    def predict_batch(self, features_batch: np.ndarray) -> np.ndarray:
        """Inférence batch. Shape (N, 24) → (N,)."""
        h1 = np.maximum(0, features_batch @ self.W1 + self.b1)
        h2 = np.maximum(0, h1 @ self.W2 + self.b2)
        out = h2 @ self.W3 + self.b3
        return 1.0 / (1.0 + np.exp(-out[:, 0]))

    def get_params(self) -> np.ndarray:
        """Flatten tous les paramètres (pour CMA-ES)."""
        return np.concatenate([self.W1.ravel(), self.b1, self.W2.ravel(),
                                self.b2, self.W3.ravel(), self.b3])

    def set_params(self, params: np.ndarray) -> None:
        """Restore depuis vecteur flat (pour CMA-ES)."""
        i = 0
        s = self.W1.size; self.W1 = params[i:i+s].reshape(self.W1.shape); i+=s
        s = self.b1.size; self.b1 = params[i:i+s]; i+=s
        s = self.W2.size; self.W2 = params[i:i+s].reshape(self.W2.shape); i+=s
        s = self.b2.size; self.b2 = params[i:i+s]; i+=s
        s = self.W3.size; self.W3 = params[i:i+s].reshape(self.W3.shape); i+=s
        s = self.b3.size; self.b3 = params[i:i+s]; i+=s

    def save(self, path: str) -> None:
        np.save(path, self.get_params())

    def load(self, path: str) -> None:
        self.set_params(np.load(path))

def evaluate_state(state: GameState, me: int, evaluator: NumpyEvaluator = None) -> float:
    """
    Score un état de jeu pour le joueur `me`.
    Si evaluator=None, utilise heuristique rapide.
    """
    if evaluator is not None:
        features = extract_features(state, me)
        return evaluator.predict(features)
    else:
        return _heuristic_eval(state, me)

def _heuristic_eval(state: GameState, me: int) -> float:
    """
    Évaluation heuristique de fallback (sans NN).
    Rapide, moins précise.
    """
    total_ships = 0
    my_ships = 0
    my_production = 0
    total_production = 0

    for p in state.planets:
        ships = p[5]
        prod = p[6]
        total_ships += ships
        total_production += prod
        if p[1] == me:
            my_ships += ships
            my_production += prod

    if len(state.fleets) > 0:
        for f in state.fleets:
            total_ships += f[6]
            if f[1] == me:
                my_ships += f[6]

    if total_ships == 0:
        return 0.5

    ship_ratio = my_ships / total_ships
    prod_ratio = my_production / max(total_production, 1)

    # Pondération : ships comptent plus en endgame, production en early
    t = state.step / 500.0
    return (1 - t) * 0.4 * prod_ratio + t * 0.6 * ship_ratio + (1-t) * 0.6 * ship_ratio + t * 0.4 * prod_ratio

def generate_self_play_games(evaluator: NumpyEvaluator,

    for game_idx in range(n_games):

6.3 ENTRAÎNEMENT PAR CMA-ES (option 1, plus simple)

def train_evaluator_cma(evaluator: NumpyEvaluator,
                         dataset: list,
                         n_generations: int = 50) -> NumpyEvaluator:
    """
    Entraîne l'évaluateur via CMA-ES sur le dataset self-play.

    Loss = Binary Cross-Entropy entre prédiction et outcome réel.
    """
    import cma

    X = np.array([d[0] for d in dataset], dtype=np.float32)
    y = np.array([d[1] for d in dataset], dtype=np.float32)

    def loss(params):
        evaluator.set_params(np.array(params, dtype=np.float32))
        preds = evaluator.predict_batch(X)
        # Binary cross-entropy
        bce = -np.mean(y * np.log(preds + 1e-8) + (1-y) * np.log(1-preds + 1e-8))
        return float(bce)

    init_params = evaluator.get_params().tolist()
    es = cma.CMAEvolutionStrategy(init_params, sigma0=0.1,
                                    {'maxiter': n_generations, 'verbose': -9})
    es.optimize(loss)
    evaluator.set_params(np.array(es.result.xbest, dtype=np.float32))
    return evaluator

6.4 ENTRAÎNEMENT PAR BACKPROP NUMPY (option 2, plus rapide)

def train_evaluator_backprop(evaluator: NumpyEvaluator,
                              dataset: list,
                              epochs: int = 100,
                              lr: float = 0.001,
                              batch_size: int = 64) -> NumpyEvaluator:
    """
    Entraîne par gradient descent (backprop numpy manuel).
    Plus rapide que CMA-ES pour de gros datasets.
    """
    X = np.array([d[0] for d in dataset], dtype=np.float32)
    y = np.array([d[1] for d in dataset], dtype=np.float32)

    for epoch in range(epochs):
        # Shuffle
        idx = np.random.permutation(len(X))
        X, y = X[idx], y[idx]

        total_loss = 0
        for i in range(0, len(X), batch_size):
            xb = X[i:i+batch_size]
            yb = y[i:i+batch_size]

            # Forward
            h1 = np.maximum(0, xb @ evaluator.W1 + evaluator.b1)
            h2 = np.maximum(0, h1 @ evaluator.W2 + evaluator.b2)
            logit = h2 @ evaluator.W3 + evaluator.b3
            pred = 1.0 / (1.0 + np.exp(-logit[:, 0]))

            # Loss BCE
            loss = -np.mean(yb * np.log(pred + 1e-8) + (1-yb) * np.log(1-pred + 1e-8))
            total_loss += loss

            # Backward (chain rule manuel)
            d_pred = -(yb / (pred + 1e-8) - (1-yb) / (1-pred + 1e-8)) / len(xb)
            d_logit = d_pred * pred * (1 - pred)

            d_W3 = h2.T @ d_logit[:, np.newaxis]
            d_b3 = d_logit.sum(axis=0)
            d_h2 = d_logit[:, np.newaxis] @ evaluator.W3.T

            d_h2_relu = d_h2 * (h2 > 0)
            d_W2 = h1.T @ d_h2_relu
            d_b2 = d_h2_relu.sum(axis=0)
            d_h1 = d_h2_relu @ evaluator.W2.T

            d_h1_relu = d_h1 * (h1 > 0)
            d_W1 = xb.T @ d_h1_relu
            d_b1 = d_h1_relu.sum(axis=0)

            # Update (Adam serait mieux mais SGD suffit pour commencer)
            evaluator.W1 -= lr * d_W1
            evaluator.b1 -= lr * d_b1
            evaluator.W2 -= lr * d_W2
            evaluator.b2 -= lr * d_b2
            evaluator.W3 -= lr * d_W3
            evaluator.b3 -= lr * d_b3

        if epoch % 10 == 0:
            print(f"Epoch {epoch}: loss={total_loss:.4f}")

    return evaluator

6.5 LEAGUE (self-play diversifiée)

class League:
    """
    Garde des snapshots historiques du bot pour éviter l'overfitting.
    Inspiré de AlphaStar league.
    """
    def __init__(self, max_size: int = 10):
        self.snapshots = []  # List of (evaluator_params, elo_estimate)
        self.max_size = max_size

    def add_snapshot(self, evaluator: NumpyEvaluator, elo: float = 1000):
        params = evaluator.get_params().copy()
        self.snapshots.append((params, elo))
        if len(self.snapshots) > self.max_size:
            # Garder les plus forts + quelques anciens pour diversité
            self.snapshots.sort(key=lambda x: -x[1])
            self.snapshots = self.snapshots[:self.max_size]

    def sample_opponent(self) -> NumpyEvaluator:
        """Échantillonne un adversaire de la league."""
        if not self.snapshots:
            return None
        params, _ = self.snapshots[np.random.randint(len(self.snapshots))]
        opp = NumpyEvaluator()
        opp.set_params(params)
        return opp

def train_v6(n_iterations: int = 20,
              games_per_iter: int = 50,
              n_jobs: int = 4):

    Boucle principale d'entraînement AlphaGo-style.

    1 iteration = générer des parties + entraîner évaluateur
    """
    evaluator = NumpyEvaluator()
    league = League()

    for iteration in range(n_iterations):
        print(f"\n=== Iteration {iteration+1}/{n_iterations} ===")

        # 1. Self-play
        print("Génération des parties self-play...")
        dataset = generate_self_play_games(evaluator, games_per_iter, n_jobs)
        print(f"  {len(dataset)} états collectés")

        # 2. Entraînement
        print("Entraînement évaluateur...")
        evaluator = train_evaluator_backprop(evaluator, dataset, epochs=50)

        # 3. Évaluation vs league
        win_rate = evaluate_vs_league(evaluator, league, n_games=20)
        print(f"  Win rate vs league: {win_rate:.1%}")

        # 4. Sauvegarder si amélioration
        elo_estimate = 800 + win_rate * 800  # rough estimate
        league.add_snapshot(evaluator, elo_estimate)
        evaluator.save(f"checkpoints/eval_iter_{iteration:03d}.npy")

        # 5. Sauvegarder meilleur
        if win_rate > 0.6:
            evaluator.save("best_evaluator.npy")
            print(f"  ✅ Nouveau meilleur sauvegardé (win_rate={win_rate:.1%})")

SECTION 9 — CONSIDÉRATIONS TECHNIQUES AVANCÉES
================================================================================

9.1 POURQUOI PAS MCTS ?

MCTS (Monte Carlo Tree Search) nécessite :
- Espace d'action discret et borné → difficile (angles continus)
- Beaucoup de simulations (10000+/tour) → trop lent sans C extension
- Implémentation complexe

Beam search est préférable ici car :
- Espace d'action naturellement continu
- Plus simple à implémenter correctement
- Plus rapide à itérer
- Suffisant pour 1400-1600 ELO

9.2 POURQUOI CMA-ES POUR L'ÉVALUATEUR ?

Alternative à backprop manuel :
- Backprop numpy = gradient exact mais fragile (learning rate sensible)
- CMA-ES = gradient-free, robuste, converge bien sur <1000 params
- Pour NumpyEvaluator (~900 params) : CMA-ES peut suffire
- Recommandation : utiliser backprop pour datasets > 10000 exemples

9.3 OPTIMISATIONS FUTURES

Si beam search atteint un plafond (~1400 ELO) :
1. Augmenter horizon de 20 à 50 tours (nécessite sim 5× plus rapide)
2. Implémenter opponent modeling (mémoriser patterns adverses)
3. Ajouter policy network (guider génération de candidats)
4. Porter le simulateur critique en Cython ou ctypes (10-100× plus rapide)
5. Implémenter PUCT (AlphaGo Zero style) si sim assez rapide
