# Legacy Archive

This folder contains archived V1-V6 code and related utilities.

## Layout

- `bots/` - old bot implementations and submission copies
- `training/` - old training scripts
- `benchmarks/` - old benchmark and timing scripts
- `artifacts/` - old evaluator weights and serialized assets
- `backups/` - old one-off backup files

The active project now lives at the repository root with V7/V8 files only.
